第二周作业

进制转换

  1. 1分12秒 = 72000 毫秒
  2. (7A)13 = (101)10
  3. (7A)16 = (122)10
  4. (7A)16 = (1111010)2 = (172)8
  5. (1011011)2 = (133)8 = (5B)16
  6. (678)10 = (1010100110)2 = (1246)8
  7. (111)2 + (101)2 = (1100)2

解释关键词

Information Information can be thought of as the resolution of uncertainty

Positional notation Positional notation denotes usually the extension to any base of the Hindu–Arabic numeral system.

algorithm In mathematics and computer science, an algorithm is a finite sequence of well-defined, computer-implementable instructions, typically to solve a class of problems or to perform a computation.

software bug A software bug is an error, flaw or fault in a computer program or system that causes it to produce an incorrect or unexpected result, or to behave in unintended ways.
